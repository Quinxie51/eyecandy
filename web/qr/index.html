Get the QR Code

Share this QR code so anyone can scan it with their phone and instantly browse and try on products in augmented reality — no app required.

Deployment steps:
1. Push the web/ folder to a public GitHub repo.
2. Enable GitHub Pages (Settings → Pages → Deploy from branch).
3. Or deploy to Vercel / Netlify by dragging the web/ folder.
4. Paste your live URL above and re-generate the QR code.
